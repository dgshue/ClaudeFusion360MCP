**construction_plane(design, rootComp, params):**
- Accept `mode` param: "offset" (default) or "angle"
- For offset mode: accept `plane` (string: "XY", "XZ", "YZ", default "XY") and `offset` (float, cm, default 1.0)
  - Map plane string to rootComp.xYConstructionPlane / xZConstructionPlane / yZConstructionPlane
  - Use `plane_input.setByOffset(base_plane, adsk.core.ValueInput.createByReal(offset))`
- For angle mode: accept `plane` (base plane string), `axis` (string: "X", "Y", "Z"), `angle` (float, degrees)
  - Map axis string to rootComp.xConstructionAxis / yConstructionAxis / zConstructionAxis
  - Map plane string to construction plane
  - Use `plane_input.setByAngle(axis_entity, adsk.core.ValueInput.createByString(f"{angle} deg"), base_plane)`
- Return `{"success": True, "name": plane.name}`

**construction_axis(design, rootComp, params):**
- Accept `mode` param: "two_points" (default) or "axis"
- For two_points mode: accept `point1` (dict with x,y,z) and `point2` (dict with x,y,z)
  - Create Point3D objects: `adsk.core.Point3D.create(x, y, z)`
  - Use `axis_input.setByTwoPoints(point1_obj, point2_obj)`
- For axis mode: accept `axis` (string: "X", "Y", "Z") -- return the existing construction axis name for reference
  - Map to rootComp.xConstructionAxis etc.
  - Actually create using `axis_input.setByLine(rootComp.xConstructionAxis)` if a new one is needed, OR just return existing axis info
  - Simplest approach: use setByTwoPoints with origin and axis endpoint for predictable behavior
- Return `{"success": True, "name": axis.name}`

**construction_point(design, rootComp, params):**
- Accept `x`, `y`, `z` (floats, cm, all default 0)
- Create Point3D: `adsk.core.Point3D.create(x, y, z)`
- Use `point_input.setByPoint(point3d)`
- Return `{"success": True, "name": point.name}`

**construction_plane tool:**

```python
@mcp.tool()
def construction_plane(mode: str = "offset", plane: str = "XY", offset: float = 1.0, axis: str = "X", angle: float = 45.0) -> dict:
    """
    Create a construction (reference) plane for sketching or feature operations.

    Args:
        mode: Creation mode - "offset" (parallel to base plane) or "angle" (rotated around axis)
        plane: Base plane - "XY", "XZ", or "YZ" (default "XY")
        offset: Distance from base plane in cm (used in offset mode, default 1.0)
        axis: Rotation axis for angle mode - "X", "Y", or "Z" (default "X")
        angle: Rotation angle in degrees (used in angle mode, default 45.0)
    """
    params = {"mode": mode, "plane": plane, "offset": offset, "axis": axis, "angle": angle}
    return send_fusion_command("construction_plane", params)
```

```python
def construction_axis(mode: str = "two_points", point1_x: float = 0, point1_y: float = 0, point1_z: float = 0, point2_x: float = 0, point2_y: float = 0, point2_z: float = 1.0) -> dict:
```
