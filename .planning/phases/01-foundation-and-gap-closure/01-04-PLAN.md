    Replace placeholder stubs for all 6 component handlers. Per user decision, coordinate correction does NOT apply to move_component/rotate_component -- they use standard Fusion coordinates.

    **create_component (COMP-01):**
    - Get the body to convert: `body = get_body(rootComp, params.get('body_index'))`.
    - Use the body-to-component approach: create new occurrence, move body into it.
    - Simpler approach: `occ = rootComp.occurrences.addNewComponent(adsk.core.Matrix3D.create())`. Then move body into the new component via cut/paste or use `rootComp.features.moveFeatures` API.
    - Alternative simpler pattern: `body.moveToComponent(occ)` if available.
    - If `params.get('name')` provided, set `occ.component.name = name`.
    - Return `{"success": True, "component_name": occ.component.name, "occurrence_index": index}`.

    **list_components (COMP-02):**
    - Iterate `rootComp.occurrences`:
      ```python
      components = []
      for i in range(rootComp.occurrences.count):
          occ = rootComp.occurrences.item(i)
          transform = occ.transform
          bbox = occ.boundingBox
          components.append({
              "index": i,
              "name": occ.component.name,
              "position": [round(transform.translation.x, 4), round(transform.translation.y, 4), round(transform.translation.z, 4)],
              "bounding_box": {
                  "min": [round(bbox.minPoint.x, 4), round(bbox.minPoint.y, 4), round(bbox.minPoint.z, 4)],
                  "max": [round(bbox.maxPoint.x, 4), round(bbox.maxPoint.y, 4), round(bbox.maxPoint.z, 4)]
              }
          })
      ```
    - Return `{"success": True, "component_count": len(components), "components": components}`.

    **delete_component (COMP-03):**
    - Use `get_occurrence(rootComp, index=params.get('index'), name=params.get('name'))`.
    - Call `occ.deleteMe()`.
    - Return `{"success": True}`.

    **move_component (COMP-04):**
    - Use `get_occurrence(rootComp, index=params.get('index'), name=params.get('name'))`.
    - If `absolute=True`: Create new Matrix3D, set translation to (x, y, z), set `occ.transform = matrix`.
    - If `absolute=False`: Get current transform, add offset to translation vector, set back.
    - Per user decision: NO coordinate correction -- use standard Fusion coordinates.
    - Return `{"success": True, "position": [x, y, z]}`.

    **rotate_component (COMP-05):**
    - Use `get_occurrence(rootComp, index=params.get('index'), name=params.get('name'))`.
    - Create rotation Matrix3D:
      ```python
      matrix = occ.transform.copy()  # Keep current position
      axis_map = {'X': Vector3D.create(1,0,0), 'Y': Vector3D.create(0,1,0), 'Z': Vector3D.create(0,0,1)}
      axis_vec = axis_map.get(params.get('axis', 'Z'))
      origin = Point3D.create(params.get('origin_x', 0), params.get('origin_y', 0), params.get('origin_z', 0))
      matrix.setToRotation(math.radians(angle), axis_vec, origin)
      occ.transform = matrix
      ```
    - Note: setToRotation REPLACES the matrix -- need to compose rotation with existing transform. Use `current_transform.transformBy(rotation_matrix)` pattern instead.
    - Return `{"success": True}`.

    **check_interference (COMP-06):**
    - Per research: use bounding box collision detection (MCP tool description says "bounding box collision detection").
    - Iterate all pairs of occurrences, check if bounding boxes overlap.
    - Bounding box overlap check: boxes overlap if they overlap on all 3 axes (min1 < max2 AND min2 < max1 for each axis).
    - Return `{"success": True, "interference_count": N, "interferences": [{occ1_name, occ2_name, overlap_volume}]}`.

<task type="auto">
  <name>Task 2: Implement 4 joint handlers</name>
  <files>fusion-addin/handlers/joint.py</files>
  <action>
    Replace placeholder stubs for all 4 joint handlers.

    **create_revolute_joint (JOINT-01):**
    - Follow the pattern from RESEARCH.md Joint Creation code example.
    - Get occurrences: if component indices provided, use `get_occurrence`. If not, use first two occurrences.
    - Create JointGeometry using `JointGeometry.createByPoint(occurrence, Point3D.create(x, y, z))` for each component.
    - Create joint input: `joints = rootComp.joints; jointInput = joints.createInput(geo0, geo1)`.
    - Set as revolute: `jointInput.setAsRevoluteJointMotion(JointDirections.ZAxisJointDirection)`.
      - Map axis params to JointDirections: axis_z=1 -> ZAxis, axis_x=1 -> XAxis, axis_y=1 -> YAxis.
    - Add joint: `joint = joints.add(jointInput)`.
    - Set limits if provided: `joint.jointMotion.rotationLimits.isMinimumValueEnabled = True; joint.jointMotion.rotationLimits.minimumValue = math.radians(min_angle)` (similarly for max).
    - Handle flip: `joint.isFlipped = params.get('flip', False)`.
    - Return `{"success": True, "joint_name": joint.name}`.

    **create_slider_joint (JOINT-02):**
    - Same pattern as revolute but use `jointInput.setAsSliderJointMotion(JointDirections.XAxisJointDirection)`.
    - Map axis from params (axis_x/y/z) to JointDirections.
    - Set distance limits if min_distance/max_distance provided.
    - Return `{"success": True, "joint_name": joint.name}`.

    **set_joint_angle (JOINT-03):**
    - Find joint by index: iterate `rootComp.joints`, get item at joint_index (default to last joint).
    - Verify it's a revolute joint: `joint.jointMotion.jointType == JointTypes.RevoluteJointType`.
    - Set angle: `joint.jointMotion.rotationValue = math.radians(params['angle'])`.
    - Return `{"success": True, "joint_name": joint.name, "angle_degrees": params['angle']}`.

    **set_joint_distance (JOINT-04):**
    - Find joint by index, verify it's a slider joint.
    - Set distance: `joint.jointMotion.slideValue = params['distance']` (in cm).
    - Return `{"success": True, "joint_name": joint.name, "distance_cm": params['distance']}`.

    **Error cases for all joint handlers:**
    - Raise descriptive errors if: no occurrences exist, joint index out of range, wrong joint type for set_angle/set_distance.
    - Error messages should follow the verbose pattern: "set_joint_angle(angle=45, joint_index=2) failed: Joint 2 is a SliderJoint, not a RevoluteJoint. Use set_joint_distance() instead."
  </action>
