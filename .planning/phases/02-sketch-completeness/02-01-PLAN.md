  <action>
Create `fusion-addin/helpers/sketch_entities.py` with these functions:

1. `get_sketch_curve(sketch, curve_index)` - Returns the SketchCurve at the given index from `sketch.sketchCurves`. Raises ValueError with message including the valid range and suggestion to use get_sketch_info(). Check `curve_index < 0 or curve_index >= sketch.sketchCurves.count`.

2. `get_sketch_point(sketch, point_index)` - Returns the SketchPoint at the given index from `sketch.sketchPoints`. Same error pattern. Note: index 0 is the origin point.

3. `get_curve_index(sketch, curve)` - Searches `sketch.sketchCurves` for the given curve object by iterating and comparing references. Returns integer index or -1 if not found.

4. `find_point_index(sketch, sketch_point)` - Searches `sketch.sketchPoints` for the given SketchPoint object. Returns integer index or -1.

5. `label_curve(curve, index)` - Generates semantic label string. Use isinstance checks:
   - SketchLine: `"Curve {index} (line, horizontal|vertical|diagonal, {length}cm)"` (horizontal if abs(dy) < 0.001, vertical if abs(dx) < 0.001)
   - SketchCircle: `"Curve {index} (circle, radius {r}cm)"`
   - SketchArc: `"Curve {index} (arc, radius {r}cm, {sweep}deg)"`
   - SketchEllipse: `"Curve {index} (ellipse, {major}x{minor}cm)"`
   - SketchFittedSpline: `"Curve {index} (spline, {n} points)"`
   - Fallback: `"Curve {index} ({classname})"` with " construction" appended if `curve.isConstruction`

Import math for angle calculations. Import adsk.core and adsk.fusion. Use the same sys.path pattern as other helpers for Fusion embedded Python compatibility.

Round all numeric values to 2 decimal places.

**Add `set_construction` handler** (NSKCH-07) to `handlers/sketch.py`:
```python
def set_construction(design, rootComp, params):
    try:
        sketch = get_sketch(rootComp)
    except ValueError as e:
        return {"success": False, "error": str(e)}
    curve = get_sketch_curve(sketch, params['curve_index'])
    is_construction = params.get('is_construction', True)
    curve.isConstruction = is_construction
    return {
        "success": True,
        "curve_index": params['curve_index'],
        "is_construction": curve.isConstruction
    }
```

```python
@mcp.tool()
def set_construction(curve_index: int, is_construction: bool = True) -> dict:
    """Toggle a sketch curve between regular and construction geometry.
    Construction geometry is used as reference but not included in profiles.

    Args:
        curve_index: Index of the curve to toggle. Use get_sketch_info() to see available curves.
        is_construction: True to make construction, False to make regular. Default True.
    """
    return send_fusion_command("set_construction", {
        "curve_index": curve_index,
        "is_construction": is_construction
    })
```
