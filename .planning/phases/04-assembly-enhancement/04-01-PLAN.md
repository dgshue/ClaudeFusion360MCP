  <action>
In `fusion-addin/handlers/joint.py`, add 5 new handler functions after the existing `set_joint_distance` function. All follow the existing revolute/slider template pattern:

1. **create_rigid_joint(design, rootComp, params)**: Simplest joint -- no axis, no limits.
   - x/y/z default to (0,0,0) per user decision (position irrelevant for rigid)
   - Call `joint_input.setAsRigidJointMotion()` (no params)
   - Return `{"success": True, "joint_name": ..., "joint_index": ..., "type": "rigid", "dof": "none (fixed)"}`

2. **create_cylindrical_joint(design, rootComp, params)**: Rotation + translation along same axis.
   - Use `_get_axis_direction(params)` for axis (same as revolute/slider)
   - Call `joint_input.setAsCylindricalJointMotion(axis_direction)`
   - Set optional limits: `min_angle`/`max_angle` via `joint.jointMotion.rotationLimits` (convert degrees to radians), `min_distance`/`max_distance` via `joint.jointMotion.slideLimits`
   - Return with `"type": "cylindrical", "dof": "rotation + translation"` and limits summary

3. **create_pin_slot_joint(design, rootComp, params)**: Rotation + sliding on different axes.
   - Per user decision: single axis param defines slot direction; rotation is perpendicular
   - Derive rotation axis: if slot is Z, use X; if slot is X, use Z; if slot is Y, use Z (first non-matching standard axis)
   - Create a helper `_get_perpendicular_axis(slot_direction)` that returns a JointDirection perpendicular to the given one
   - Call `joint_input.setAsPinSlotJointMotion(rotation_axis, slot_direction)`
   - Set optional limits: `min_angle`/`max_angle` for rotation, `min_distance`/`max_distance` for slide
   - Return with `"type": "pin_slot", "dof": "rotation + slide"`

4. **create_planar_joint(design, rootComp, params)**: 2D sliding + rotation on a plane.
   - Axis param defines plane normal (e.g., Z normal = XY plane sliding)
   - Call `joint_input.setAsPlanarJointMotion(normal_direction)`
   - Set optional limits with distinct param names: `min_primary_slide`/`max_primary_slide`, `min_secondary_slide`/`max_secondary_slide`, `min_angle`/`max_angle` via `primarySlideLimits`/`secondarySlideLimits`/`rotationLimits`
   - All unconstrained by default per user decision
   - Return with `"type": "planar", "dof": "2D slide + rotation"`

5. **create_ball_joint(design, rootComp, params)**: Spherical rotation, no axis param per user decision.
   - Use default `ZAxisJointDirection` for pitch, `XAxisJointDirection` for yaw (internal reference frame)
   - Call `joint_input.setAsBallJointMotion(pitch_dir, yaw_dir)`
   - Optional limits: `min_pitch`/`max_pitch`, `min_roll`/`max_roll`, `min_yaw`/`max_yaw` via `pitchLimits`/`rollLimits`/`yawLimits`
   - Return with `"type": "ball", "dof": "spherical rotation (3 DOF)"`

6. **Extend set_joint_angle**: Change the type check from `== RevoluteJointType` to checking membership in a set `{RevoluteJointType, CylindricalJointType, PinSlotJointType}`. Update error message to say "not a joint with rotation DOF" and list valid types. The `joint.jointMotion.rotationValue = math.radians(angle)` line stays the same (all three types have `rotationValue`).

7. **Extend set_joint_distance**: Change the type check from `== SliderJointType` to checking membership in a set `{SliderJointType, CylindricalJointType, PinSlotJointType}`. Update error message accordingly. The `joint.jointMotion.slideValue = distance` line stays the same (all three types have `slideValue`). Do NOT include PlanarJointType -- it has `primarySlideValue`/`secondarySlideValue` not `slideValue`.

1. **create_rigid_joint**: Minimal params -- no axis needed, position defaults to (0,0,0).
   ```python
   @mcp.tool()
   def create_rigid_joint(
       component1_index: int = None,
       component2_index: int = None,
       x: float = 0, y: float = 0, z: float = 0
   ) -> dict:
       """Create a rigid (fixed) joint between two components. No relative motion allowed.
       Position defaults to origin since rigid joints lock components in place."""
   ```

2. **create_cylindrical_joint**: Same params as revolute + slider combined (axis + angle limits + distance limits).
   ```python
   @mcp.tool()
   def create_cylindrical_joint(
       component1_index: int = None,
       component2_index: int = None,
       x: float = 0, y: float = 0, z: float = 0,
       axis_x: float = 0, axis_y: float = 0, axis_z: float = 1,
       min_angle: float = None, max_angle: float = None,
       min_distance: float = None, max_distance: float = None
   ) -> dict:
       """Create a cylindrical joint (rotation + translation along same axis)."""
   ```

3. **create_pin_slot_joint**: Axis defines slot direction; rotation is perpendicular.
   ```python
   @mcp.tool()
   def create_pin_slot_joint(
       component1_index: int = None,
       component2_index: int = None,
       x: float = 0, y: float = 0, z: float = 0,
       axis_x: float = 0, axis_y: float = 0, axis_z: float = 1,
       min_angle: float = None, max_angle: float = None,
       min_distance: float = None, max_distance: float = None
   ) -> dict:
       """Create a pin-slot joint (rotation + sliding). Axis defines slot direction; rotation is perpendicular."""
   ```

4. **create_planar_joint**: Axis defines plane normal, distinct limit param names for 3 DOF.
   ```python
   @mcp.tool()
   def create_planar_joint(
       component1_index: int = None,
       component2_index: int = None,
       x: float = 0, y: float = 0, z: float = 0,
       axis_x: float = 0, axis_y: float = 1, axis_z: float = 0,
       min_primary_slide: float = None, max_primary_slide: float = None,
       min_secondary_slide: float = None, max_secondary_slide: float = None,
       min_angle: float = None, max_angle: float = None
   ) -> dict:
       """Create a planar joint (2D sliding + rotation on a plane). Axis defines plane normal. All DOF unconstrained by default."""
   ```

5. **create_ball_joint**: No axis param per user decision. Optional pitch/roll/yaw limits.
   ```python
   @mcp.tool()
   def create_ball_joint(

       min_roll: float = None, max_roll: float = None,
       min_yaw: float = None, max_yaw: float = None
   ) -> dict:
       """Create a ball joint (spherical rotation, 3 DOF). No axis needed -- rotation is unconstrained in all directions."""
   ```

Each tool body follows the same pattern: build params dict, add optional params if not None, call `send_fusion_command("tool_name", params)`.
